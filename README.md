# C# Data Structures

Repositório dedicado ao estudo de estruturas de dados, algoritmos e resoluçăo de problemas utilizando C#.

Este projeto foi criado para documentar minha evoluçăo na resoluçăo de coding challenges, praticando padrőes de algoritmos e estruturas de dados frequentemente utilizados em entrevistas técnicas.
